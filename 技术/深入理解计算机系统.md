## 一、内容简介

本书从程序员的视角全面介绍了计算机系统的底层原理，包括数据的机器级表示、处理器体系结构、程序的机器级表示、存储器层次结构、链接、异常控制流、虚拟存储器以及系统级 I/O 等内容，帮助读者深入理解程序是如何在计算机硬件上运行的。

---

## 二、核心观点 / 主要内容

### 第一章：计算机系统漫游

- 信息即位 + 上下文：所有信息（数字、字符串、程序）都以比特表示，意义取决于上下文。
- 程序的生命周期：源码 → 预处理 → 编译 → 汇编 → 链接 → 可执行文件。
- 存储器层次结构：寄存器、L1/L2/L3 缓存、主存、磁盘，越往上越快越小越贵。

### 第二章：信息的表示和处理

- 整数的补码表示使加减法电路统一，是现代 CPU 的基础。
- 浮点数遵循 IEEE 754 标准，存在精度损失和舍入误差。

### 第六章：存储器层次结构

- 局部性原理（时间局部性 + 空间局部性）是缓存能够发挥作用的根本原因。
- 理解缓存命中/缺失对编写高性能代码至关重要。

## 四、读后感

本书是理解计算机底层运作的经典之作，对于想要写出高性能代码、深入理解操作系统和编译器的程序员来说是必读书目。书中将抽象的硬件概念与具体的 C 程序示例紧密结合，让理论不再枯燥。读完之后，对缓存优化、内存对齐、系统调用等话题有了更直观的认识。
