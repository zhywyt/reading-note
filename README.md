# 读书笔记 / Reading Notes

用于存储个人读书笔记的仓库。

## 目录结构 / Directory Structure

```
reading-note/
├── README.md               # 本文件 / This file
├── TEMPLATE.md             # 读书笔记模板 / Note template
├── 技术/                   # 技术类书籍 / Technical books
├── 文学/                   # 文学类书籍 / Literature
├── 历史/                   # 历史类书籍 / History
├── 哲学/                   # 哲学类书籍 / Philosophy
└── 其他/                   # 其他类书籍 / Other
```

## 使用方法 / How to Use

1. 复制 `TEMPLATE.md` 到对应分类目录，以书名命名（例如 `技术/深入理解计算机系统.md`）。
2. 按模板格式填写读书笔记。
3. 在下方的书单中记录已读书目。
